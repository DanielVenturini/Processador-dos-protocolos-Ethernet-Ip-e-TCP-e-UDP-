PROCESSADOR DOS CABEÇALHOS DOS PROTOCOLOS ETHERNET IP TCP E UDP

Usando este programa https://gist.github.com/austinmarton/2862515 para capturar os pacotes via socket, estes são salvos em hexadecimal. Então o meu programa converte esse arquivo para binário e imprime os valores dos cabeçalhos - em decimal. Porém, os arquivos binarios originais precisam sem convertidos devido a sua formatacao. Um modo de fazer isso, é abrir o arquivo no Sublime Text, onde aparecem os Bytes do arquivo. Apos, salvar o arquivo.
